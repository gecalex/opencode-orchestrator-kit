# MCP Analyzer

Анализирует ключевые слова в ТЗ/спецификациях и подбирает подходящие MCP сервера.

## Ключевые слова → MCP

### Базы данных
| Ключевые слова | MCP |
|---------------|-----|
| postgres, postgresql, sql, база данных | postgres |
| sqlite, файл база | sqlite |
| supabase | supabase |
| neo4j, граф, graph | neo4j |
| mysql | mysql |

### Поиск
| Ключевые слова | MCP |
|---------------|-----|
| поиск, искать, найти | searxng |

## Использование

### Шаг 1: Прочитать ТЗ
```
read: SPEC.md
```

### Шаг 2: Вызвать mcp-analyzer
```
speckit-mcp-analyze --spec "SPEC.md"
```

### Шаг 3: Получить рекомендацию
Система вернёт:
- Категорию задачи
- Список MCP серверов
- Команды для настройки

## Примеры

### Пример 1: Веб-приложение на Python
```
ТЗ: Создать веб-приложение на Flask с БД PostgreSQL

Ключевые слова обнаружены:
- web, flask → http сервер
- postgresql → postgres MCP

Рекомендуемые MCP:
- filesystem (работа с файлами)
- postgres (база данных)
```

### Пример 2: API с автотестами
```
ТЗ: REST API с автотестами

Рекомендуемые MCP:
- filesystem
- postgres (если есть БД)
- playwright (если UI тесты)
```
