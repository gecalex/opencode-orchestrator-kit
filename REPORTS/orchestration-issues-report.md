## 1. ДОКУМЕНТАЦЫЯ VS РЕАЛЬНОСТЬ

### Документировано в AGENTS.md

| Команда | Описание |
|---------|-----------|
| `speckit-task-delegator` | Делегирование задачи агенту |
| `speckit-quality-gate-run` | Запуск Quality Gate проверок |
| `speckit-analyze-state` | Анализ состояния |

### Реальность

**Эти команды НЕ существуют как CLI инструменты!** Они задокументированы, но недоступны.

```bash
# Пытался вызвать:
speckit-task-delegator --prompt "..." --subagent_type "python-developer"
# Результат: Command not found
```

## 3. КОНКРЕТНЫЕ ПРИМЕРЫ

### Пример 1: Исправление backtesting

**Как было (я делал сам):**
- Нашёл ошибку в `engine.py`
- Отредактировал файл
- Запустил тесты

**Как должно было быть:**
```
task --subagent_type python-developer --prompt "Исправь NameError: name 'timezone' is not defined в services/backtest/src/engine.py. Запусти тесты после исправления."
```

### Пример 2: NATS Integration

**Как было (я делал сам):**
- Написал 100+ строк Go кода
- Проверил компиляцию

**Как должно было быть:**
```
task --subagent_type go-developer --prompt "Добавь NATS integration в internal/connector/nats_integration.go для публикации market data событий"
```

## 4. ПРИЧИНЫ ПРОБЛЕМЫ

### А. Документация вводит в заблуждение

```
AGENTS.md говорит:
- "speckit-task-delegator — Делегирование задачи агенту"
- "speckit-quality-gate-run — Выполнение Quality Gate"

Но эти команды НЕ существуют!
```

### Б. Нет явного Trigger для агентов

В Task tool есть все агенты:
- `python-developer`
- `go-developer`  
- `react-developer`
- `code-reviewer`
- `security-auditor`

Но **нет документации** как их вызывать кроме как через Task tool.

### В. Мой опыт

Я привык делать сам, потому что:
1. Задачи казались "простыми" (5-15 минут)
2. Не было явной инструкции "ДЛЯ ЭТОГО используй агента X"
3. Git workflow уже освоен

## ЗАКЛЮЧЕНИЕ

**Проблема не в отсутствии инструментов, а в:**
1. Документация обещает несуществующие команды
2. Нет явных правил делегирования
3. Моя привычка делать самому

**Решение:**
- Использовать Task tool для всех кодовых задач
- Добавить правило в AGENTS.md: "ВСЕГДА используй специализированного агента для кода, если задача > 15 минут"
