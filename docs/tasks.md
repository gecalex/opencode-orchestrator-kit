1. Выполняйте задачи в порядке возрастания ID внутри каждой фазы.
2. После выполнения каждой задачи обновляйте поле **Статус** на **completed**.
3. После завершения всей фазы проверить, что все критерии завершения фазы выполнены, и только тогда переходить к следующей фазе.
4. При возникновении блокировок или необходимости уточнения — создавайте дополнительные подзадачи или обсуждайте с заинтересованными сторонами.
5. Весь прогресс фиксируйте коммитами в ветке `develop` с описательными сообщениями, ссылающимися на ID задачи (например: `feat: TASK-102 - добавлен диалог подтверждения инициализации Git`).
